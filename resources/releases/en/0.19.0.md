# 0.19.0

Date: 2025-11-05

## What’s new

- Installable app: you can now install the game from supported browsers (Chrome/Edge) — use the browser "Install" menu or "Add to Home screen" on mobile to run the game in a separate window with its own icon.
- Daily population summary: the daily population tick now creates a single summary Notification showing how many people were born and how many died during that tick.

Help: updated help pages with brief instructions about installing the app and where to find the daily population summary (English and Bulgarian).
